# CafItamita — App de pedidos de café antes de clase

Repositorio del proyecto final del curso de **Análisis y Diseño de Sistemas de Información (ADSI)** — Entrega: 29/05/2026.

---

## Descripción del proyecto

**CafItamita** es una aplicación móvil que permite a estudiantes y docentes programar su pedido de café antes de entrar a clase, eliminando las filas en la cafetería y optimizando su tiempo entre clases.

La app cubre los cuatro requisitos funcionales del proyecto:

| Requisito | Implementación en CafItamita |
|---|---|
| Seleccionar el menú | Catálogo con grid de productos, barra de búsqueda inferior (one-handed) y saludo personalizado |
| Elegir un pedido | Tarjetas de producto con botón de agregar; carrito con contador dinámico |
| 3 opciones de pago | Apple Pay / ID, tarjeta de crédito/débito (Mastercard) y PayPal |
| Confirmar hora de entrega | Time picker con selector de hora de recogida integrado en el resumen de orden |
